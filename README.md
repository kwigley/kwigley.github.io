# [kylewigs.com](https://kylewigs.com)

Built with Astro and deployed to GitHub Pages.

### Getting Started

Install dependencies: `pnpm install`

Run dev server locally: `pnpm dev`

Format code: `pnpm format`

Check types: `pnpm check`
